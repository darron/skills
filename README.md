# Darron's Codex Skills

Personal Codex skill backups and portable project skills.

## Skills

- `claude-code-review`: adversarial Claude review workflow for implemented code.
- `claude-doc-review`: adversarial Claude review workflow for documents, plans, and specs.
- `doc-review-merge`: merge reviewer feedback into one focused document.
- `go-vite-svelte-builder`: planning and implementation guardrails for Go-routed Vite/Svelte apps.
- `panopticon-codex-upload`: summarize and upload Codex session notes to Panopticon.
- `research-pulse`: run Converge Research Pulse workflows through the supported CLI/API contract.

## Excluded

These are intentionally not vendored here:

- Codex `.system` skills.
- Plugin cache skills from OpenAI-bundled, OpenAI-curated, or OpenAI primary runtime plugins.
- Anthropic example and document skills from `anthropics/skills`.
- `dbrain-mcp`, because it already lives in the open-source `dbrain` repo.
- `ff-workers` skills, because they already live in their open-source repo.
- `odysseus`, per local publication scope.
- Third-party skills with explicit external ownership, such as `last30days`.
